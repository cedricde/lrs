This tool is a wrapper around OCSinventory to send the results
of a software inventory to a single server via HTTP. 

That server must run the LRS Inventory Webmin module (available on
ftp.linbox.com).

For more information, read the LRS documentation:

  http://www.linbox.com/en/docs/ch06.html

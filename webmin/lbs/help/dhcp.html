DHCP
This form is a simple way to manually append a new host in the DHCP configuration file.

The Name field is the refered computer name. This name should be composed of alphanumeric characters only (ranges [a-z], [A-Z] and '-').

The MAC address field must be the host network card's physical address. It shoud be a string made of 6 hexa digits, and delimited with semi-colons ':' . Example: 01:aa:45:e4:05:26 .

The IP address field is optional. If empty, a valid address will be automatically selected. If you fill it, you must provide a standard numeric address. You can also enter Dynamic if your DHCP server is not set up for assigning static IP addresses.

The Admin ID field the password needed to allow this registration.

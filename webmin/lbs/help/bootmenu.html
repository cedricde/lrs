Boot Menu
The form on this page is a representation of what the boot menu will be at start time of the computer.

In the Default choice column, you can choose to boot a default image if the user doesn't select anything.

The Display column determines availables choices in the boot menu. So, selected items will be displayed next time the user restarts his computer.

In the Menu column are the titles of available disk images. You can change an image title by clicking on it.

In the Description column are brief descriptions of disk images. You can change an image description by clicking on it.

Use the Images... button if you want to modify the boot menu contents.

Use the Rename button to rename a host or to move it to another group.

Use the Change MAC address button if you have replaced the network card of the client in order to get your old boot menu and related backups. (Remember that the LRS relies on the client's MAC address to identify it)

Use the Remove this client button if you want to remove the current computer. It removes all data about this host.

Don't forget to click on the Apply button in order to update the boot menu.

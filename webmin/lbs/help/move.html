Images
The aim of this screen is to move/remove disks images between boot menu, local directory and the directory containing base images (eg. shared images).

The Boot menu form shows images which are currently associated with the items of the host's boot menu.

The Local images form shows images owned by the current host, but which are not used in the boot menu.

The Base images form contains current base images available to all registered hosts. These are named 'base' because they can be used in any host's boot menu.

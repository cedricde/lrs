Images de base

Cette page est accessible depuis la page d'accueil du module. Elle contient la liste de toutes les images de base du système, c'est à dire les images qui sont mises en partage pour toutes les machines actuellement déclarées sur le serveur LRS.

Description des colonnes:
  • Image de base
    Noms des répertoires hébergeant les images sur le disque du serveur LRS. Ceux-ci sont localisés dans (basedir)/imgbase .
  • Titre
    Titre de l'image tel qu'il apparaît dans le menu de démarrage d'une machine.
  • Description
    Courte description concernant son contenu.
  • Effacer
    Permet d'effacer complètement l'image en question. Notez qu'il n'est pas possible d'effacer une image si celle-ci est actuellement utilisée dans le menu d'une ou plusieurs machines (voir colonne Utilisée par). Si c'est la cas, vous devez d'abord supprimer le nom de cette image du menu de chaque machine l'utilisant.
  • Utilisee par
    Affiche les noms de machines utilisant les images de base dans leur menu. Notez que chaque nom de cette colonne est un lien hypertexte. Ces liens sont des raccourcis qui vous envoient directement aux pages de configuration des machines utilisatrices.
  • Conv en Locale
    Permet de convertir une image partagée en image locale, si elle n'est pas utilisée par plus d'un client.

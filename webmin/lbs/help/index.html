LRS - Linbox Rescue Server

LRS is a client/serveur application which can backup and restore remote PC hard disks across the local network.

This module is a simple interface to configure backup and restore behavior of each computer using LRS services.

LRS Homepage

On LRS homepage, an array of registered hosts is displayed. It contains a list of all hosts which have been registered on the LRS server. Each hostname in the name column is a link on its b>boot menu configuration.

Base images
This page show a listing of all base images currently registered. In other words, all images shared between client hosts are displayed here.

The Base image column displays directory names associated with each image on the LRS server. These are located in (basedir)/imgbase .

The Title column shows base images title. The same titles which appear in a client boot menu.

The Description column is a brief image description.

The Used by column is useful to know which host(s) are currently using a particular image in its boot menu. Note that each host name (if any) in this column is a link to this host local images configuration.

In the Remove column are links to completely erase one or more images. Note that you can't remove an image if it is currently used in a host's boot menu. In order to do that, you must first delete the image title from the host's boot menu.

The Conv to Local column allow you to convert a shared image to a local one if it's not used by more than one client.
